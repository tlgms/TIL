# 인터페이스 선택

> 💡 윈도우 명령어: `route PRINT`

```mermaid
flowchart TB
    subgraph U["user mode"]
        CH["Chrome"]
        SOCK(("socket"))
    end
    subgraph K["kernel mode"]
        TCP["TCP"]
        IP["IP"]
    end
    subgraph HW["H/W"]
        D1["Driver"]
        D2["Driver"]
        NIC["NIC (유선)"]
        WL["무선"]
    end

    CH --- SOCK --> TCP --> IP
    IP --> D1 --> NIC
    IP --> D2 --> WL
    NIC --> NET(("Internet"))
    WL --> NET

    IP -. "switching =<br/>interface 선택" .- HW
```

- 왼쪽 참고 스택: HTTP / SSL / TCP / IP (u–k–H/W 경계)

### 메모

- **OS / Network stack / Routing manager**가 **메트릭 비용을 기준으로** (인터페이스를) 선택한다.
